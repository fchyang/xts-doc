# STS

## 注意事项

* STS跑测使用userdebug image
* server端需要下载ghrida，如果没有预先准备，STS tool会进行下载，但是很慢，推荐手动下载，地址：[https://github.com/NationalSecurityAgency/ghidra/releases](https://github.com/NationalSecurityAgency/ghidra/releases) ,将下载的zip压缩包拷贝到/tmp下面的同名目录，比如ghidra_11.3.2_PUBLIC_20250415.zip，就拷贝到/tmp/ghidra_11.3.2_PUBLIC_20250415.zip/ghidra_11.3.2_PUBLIC_20250415.zip
